# Europe Family Trip 2026 — Interactive Itinerary

An interactive React web app with daily OpenStreetMap maps for your Jul 1–12, 2026 Europe trip.

## What's Included

- **13 days** (Day 0–12) with hour-by-hour schedules
- **Interactive OpenStreetMap** for every day showing numbered stops and route lines
- **Sidebar navigation** grouped by destination (Disney, Paris, Europa-Park, Amsterdam, London)
- **Clickable map markers** with time and details on each stop
- **Full cost summary** with grand total, point requirements, and breakdown by category
- **Amsterdam** (Day 8–9) at Andaz Prinsengracht — Category 7 Hyatt on points

## Trip Route

```
PIT → CDG → Disneyland Paris → Paris → Europa-Park → Amsterdam → London → PIT
```

## Requirements

- Node.js 18+ ([download here](https://nodejs.org))
- npm (included with Node.js)
- Internet connection (to load OpenStreetMap tiles)

## Running Locally

```bash
# 1. Install dependencies
npm install

# 2. Start the development server
npm run dev

# 3. Open in browser
# → http://localhost:5173
```

## Building for Production

```bash
npm run build
npm run preview  # preview the built version locally
```
